## Blog ‚Äî Publishing a Post

1. Create a new file in `content/posts/` named `[slug].json` (slug = URL-friendly title, lowercase, hyphens)
2. Fill in the post JSON (see `content/posts/invertir-lotes-yucatan-2026.json` as reference)
3. Run: `node build.js`
4. Commit the updated `blog/` directory and `sitemap.xml`

## rh-agente Integration

Three integration points are pre-wired but commented out. Uncomment the one you activate first.

### 1. WhatsApp CTA (floating button + inline buttons)
In `js/main.js` ‚Äî uncomment the `initWhatsAppFloat()` block and its CSS comment.
All "Cotizar" buttons already have a commented `href` pointing to `+52 999 370 8117`.

### 2. Embedded Chat Widget
In `index.html` ‚Üí `#contacto` section ‚Äî uncomment the `<div id="rh-chat-widget">` block.
In `js/main.js` ‚Äî uncomment `initChatWidget()` and set `AGENT_WIDGET_URL`.

### 3. Contact Form ‚Üí CRM Webhook
In `index.html` ‚Üí `#contacto` section ‚Äî uncomment the `<form id="lead-form">` block.
In `js/main.js` ‚Äî uncomment `initLeadForm()` and set `CRM_WEBHOOK_URL`.

## Deploying

This is a pure static site ‚Äî no build step required for the main site. Just serve the `joelgruporh/` folder from any static host (Netlify, Vercel, GitHub Pages, Cloudflare Pages, Apache/Nginx).

For clean blog URLs (`/blog/gran-verona/` instead of `/blog/gran-verona/index.html`) to work without explicit `index.html`, most hosts handle directory indexes automatically. If using Nginx, add:

```nginx
try_files $uri $uri/ $uri/index.html =404;
```
